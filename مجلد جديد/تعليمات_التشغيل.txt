========================================
منصة الاستشارات الطبية
Medical Consultation Platform
========================================

🚀 طرق التشغيل:

الطريقة الأولى (الأسهل):
1. انقر نقراً مزدوجاً على ملف: START_HERE.bat
2. انتظر حتى يتم تثبيت التبعيات
3. سيفتح المتصفح تلقائياً على localhost:3000

الطريقة الثانية (يدوي):
1. افتح Command Prompt أو Terminal
2. انتقل إلى مجلد المشروع
3. شغل الأمر: npm install
4. شغل الأمر: npm run dev
5. افتح المتصفح على: http://localhost:3000

========================================

🔑 بيانات الدخول التجريبية:

حساب طبيب:
- البريد الإلكتروني: doctor@example.com
- كلمة المرور: demo123
- الميزات: الاستشارات، المكالمات، المراسلة

حساب مدير:
- البريد الإلكتروني: admin@example.com
- كلمة المرور: admin123
- الميزات: إدارة الأطباء، الإحصائيات، الإعدادات

========================================

📋 ميزات النظام:

✅ نظام الاستشارات الطبية الشامل
   - إنشاء وإدارة الاستشارات
   - تصنيف حسب التخصص والأولوية
   - نظام الموافقة والرفض

✅ المكالمات المرئية عبر WebRTC
   - مكالمات فيديو عالية الجودة
   - مشاركة الشاشة
   - تسجيل المكالمات (اختياري)

✅ المكتبة الطبية التفاعلية
   - رفع وتصنيف المستندات
   - نظام التقييم والمراجعات
   - البحث المتقدم

✅ نظام المراسلة الآمن
   - مراسلة مباشرة بين الأطباء
   - إشعارات القراءة
   - مشاركة الملفات

✅ تقييم الأطباء والخدمات
   - تقييم متعدد المعايير
   - عرض التقييمات على الملفات الشخصية
   - تحليل الأداء

✅ لوحة تحكم الإدارة
   - إدارة حسابات الأطباء
   - إحصائيات شاملة
   - نظام الدعم الفني

✅ واجهة عربية كاملة
   - تخطيط RTL كامل
   - خطوط عربية محسنة
   - تصميم متجاوب لجميع الأجهزة

========================================

🛠️ المتطلبات:

- Node.js الإصدار 18 أو أحدث
- npm (مدير الحزم)
- متصفح حديث (Chrome, Firefox, Edge, Safari)
- اتصال بالإنترنت (لتحميل التبعيات فقط)

========================================

🔧 استكشاف الأخطاء:

مشكلة: النظام لا يبدأ
الحل: 
- تأكد من تثبيت Node.js
- شغل: npm install
- شغل: npm run dev

مشكلة: المنفذ 3000 مستخدم
الحل:
- أغلق أي تطبيق يستخدم المنفذ 3000
- أو شغل: npm run dev -- -p 3001

مشكلة: خطأ في التبعيات
الحل:
- احذف مجلد node_modules
- شغل: npm install
