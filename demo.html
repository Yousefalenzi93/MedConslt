تعليمات التشغيل

للتشغيل السريع:

  1. 1 افتح Terminal أو Command Prompt
  2. 2 انتقل إلى مجلد المشروع
  3. 3 شغل الأمر: npm run dev
  4. 4 افتح المتصفح على: localhost:3000

للاستخدام:

  1. 1 انقر على "تسجيل الدخول"
  2. 2 استخدم أحد الحسابات التجريبية أعلاه
  3. 3 استكشف جميع ميزات النظام
  4. 4 جرب إنشاء استشارة جديدة

ملاحظة: هذا نظام تجريبي للعرض. يعمل بدون الحاجة لإعداد Firebase للاختبار.
