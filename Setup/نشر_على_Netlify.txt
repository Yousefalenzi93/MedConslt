========================================
نشر منصة الاستشارات الطبية على Netlify
https://mtmconsult.netlify.app
========================================

🚀 خطوات النشر السريع:

الخطوة 1: بناء المشروع
------------------------
انقر نقراً مزدوجاً على أحد هذه الملفات:
• build-for-netlify.bat
• build-for-netlify.ps1

أو استخدم الأوامر:
cd medical-consultation-platform
npm install
npm run build

الخطوة 2: رفع المشروع
---------------------
بعد البناء الناجح، ستجد مجلد "out"

طريقة 1 - Drag & Drop (الأسهل):
1. اذهب إلى https://netlify.com
2. سجل الدخول أو أنشئ حساب
3. اسحب مجلد "out" إلى منطقة الرفع
4. انتظر حتى يكتمل الرفع
5. غير اسم الموقع إلى "mtmconsult"

طريقة 2 - GitHub (مُوصى بها):
1. ارفع المشروع إلى GitHub
2. في Netlify: "New site from Git"
3. اختر GitHub واختر المستودع
4. اضبط الإعدادات:
   - Build command: npm run build
   - Publish directory: out
   - Node version: 18

طريقة 3 - Netlify CLI:
1. npm install -g netlify-cli
2. netlify login
3. netlify deploy --prod --dir=out

========================================

⚙️ إعدادات مهمة في Netlify:

في Site settings > Build & deploy:
• Build command: npm run build
• Publish directory: out
• Node version: 18

في Site settings > Domain management:
• Site name: mtmconsult
• Custom domain: mtmconsult.netlify.app
